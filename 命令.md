# 安装及使用
- 1、安装
```shell
npm i -g gitbook-cli
```
- 2、验证
```shell
gitbook -V
```
- 3、初始化项目
```shell
cd /path/to/project
gitbook init
```
- 4、运行服务
```shell
git serve
```
- 5、构建
```shell
# 在项目目录生成`_book`目录，生成的html文件存放在下边
git build
```

- 6、安装插件
```shell
gitbook install
```
